# homebrew-tap

A Homebrew [tap](https://docs.brew.sh/Taps) for installing Crossplane tools.

## Currently Supported Formulae

- [crossplane](https://docs.crossplane.io/latest/cli/): the official Crossplane CLI

    ```bash
    brew install tampakrap/tap/crossplane
    ```

- [xprin](https://github.com/crossplane-contrib/xprin): a testing framework for Crossplane

    ```bash
    brew install tampakrap/tap/xprin
    ```

- [xprin-helpers](https://github.com/crossplane-contrib/xprin/blob/main/docs/xprin-helpers.md): Helper standalone tools used by xprin

    ```bash
    brew install tampakrap/tap/xprin
    ```

[tap]: https://docs.brew.sh/Taps
[crossplane]: https://docs.crossplane.io/latest/cli/
